# Wasal Project 🌍

Donation and coupon distribution platform for families in need.

## 📌 Overview

Wasal is a full-stack platform built to help organizations manage:

- Donations
- Monthly coupons
- Beneficiaries
- Notifications
- QR coupon redemption
- Admin dashboard

The system provides an organized workflow for distributing aid fairly and efficiently.

# ✨ Main Features

- Secure JWT login
- Role-based access (Admin / Beneficiary)

## Coupons System
- Generate monthly coupons
- QR code redemption
- Coupon tracking
- Coupon statistics

## Donations
- Manage donations
- Track aid distribution

## Notifications
- Real-time notification system
- Mark notifications as read

## Dashboard
- Admin management panel
- Beneficiaries monitoring
- Donations overview
